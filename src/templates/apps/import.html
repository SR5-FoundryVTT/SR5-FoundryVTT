Chummer Import

  1. Enter career mode in Chummer
  2. Select language English in Chummer
  3. In Chummer... Click File then Export, select JSON in the drop down, then copy all text from the box
  4. Paste the text you copied into this box, check the appropriate boxes, then click import
Uncheck items to not import. ONLY IMPORT CHARACTER ONCE.
